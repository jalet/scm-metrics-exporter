The operator is now watching for custom resources. Create one to start exporting metrics, for example:

  apiVersion: scm.jalet.io/v1alpha1
  kind: GitHubMetricsExporter
  metadata:
    name: my-org
    namespace: {{ .Release.Namespace }}
  spec:
    org: my-org
    authMode: token
    tokenKey: token
    credentialsSecret:
      name: my-org-credentials   # a Secret with key "token" = a GitHub PAT
    export:
      otlpEndpoint: http://otel-collector.observability:4318

The operator discovers the target's repositories on spec.discoveryInterval and dispatches
one run-once collection Job per repository (capped by spec.parallelism). Each Job pushes
its metrics to spec.export.otlpEndpoint over OTLP, so point that at an OTLP collector.
